# Overview

Краткая wiki по паттерну **LLM Wiki** Андрея Карпаты для Hermes Home Server.

## Суть
Знания не «достаются» из сырых файлов каждый раз (классический RAG), а **компилируются** в устойчивые markdown-страницы: сущности, концепции, сводки, кросс-ссылки.

## Слои
- `raw/` — неизменяемые источники
- `pages/` — страницы, которые ведёт агент
- `index.md` + `log.md` — навигация и журнал

## Как пользоваться
1. В Telegram: «ingest: …» или «добавь в wiki …»
2. Вопрос: «что мы знаем про X?» (агент читает index → pages)
3. Статус без LLM: `/wikistatus`
4. Поиск без LLM: `/wikisearch` (usage; поиск с аргументами — через NL или `wiki-search.sh` в контейнере)

См. skill `wiki-llm` и [[SCHEMA]] (файл `SCHEMA.md` в корне wiki).
